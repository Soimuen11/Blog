My Dearest Mad-Readers,

Tech youtubers speak a lot about gaming on their respective operating systems. However, it is my opinion that we do not mention emulators enough. Today, I would like to present several of them to you, among which you will find one for playing:

  • PSP
  • Nintendo DS
  • Gameboy Advance
  • Game Cube
  • Wii

Installation & Configuration

The good news is that there is almost no configuration necessary. Of course, if you wish to add a controller it might be slightly more difficult than plugging it in and launching the game, but barely any more than that. Indeed, you might have to press an extra button (Gosh! This is so hard!) or set up the controller according to your personal preference in-game, but this is nothing you would not do on a Windows system.

Regarding the installation of all these emulators, they should be available in most, if not all, of your package managers. I checked on Arch Linux and Gentoo and listed the various installation commands.

Gentoo:

  • PSP: sudo emerge -a --verbose ppsspp
  • NDS: sudo emerge -a --verbose desmume
  • GBA: sudo emerge -a --verbose vbam
  • Wii / Game Cube: sudo emerge -a --verbose dolphin-emu

Arch Linux:

  • PSP: sudo pacman -S ppsspp
  • NDS: sudo pacman -S desmume
  • GBA: sudo pacman -S vbam
  • Wii / Game Cube: sudo pacman -S dolphin-emu

I would only like to add a quick word of warning regarding ppsspp: You will find 2 executable files on your system. One is called PPSSPPQt and the other is called PPSSPPHeadless. I insist: the file names are in capital letters. They are not lower cases! Do remember this if you cannot find ppsspp when typing in in dmenu or in your terminal emulator.

Other Emulators

Regarding other consoles, I advise you not to nourish too much hope about them for the time being. You will not find an emulator for PS3 or PS4 or XBOX One adapted to a Linux system. You could find something for a PS1 or PS2 and there are some games worth playing, so do feel free to set that up. However do not be delusional. Just buy the console if you are not willing to wait 10 more years for the release of an emulator.

I Have Turned My ThinkPad Into A Console

I now have many emulators installed, as well as a decent library of games. Nothing too fancy but I even scripted a simple game launcher in bash.

I also have Steam installed and a bunch of games which run natively on Linux, which is most appreciated. Of course, Steam has nothing to do with emulators… But still, I figured it might be worth mentioning.
